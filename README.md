# 🧠 About the Project

**Gojo Assistant** is a **Telegram automation bot** designed to help manage communities, store media, and provide instant file retrieval through a smart filtering system.

It combines **efficient Telegram API usage, database-backed storage, and scalable deployment support**, making it ideal for:

* 📂 File index bots
* 🤖 Assistant bots
* 👥 Community automation
* ☁️ Cloud deployments

---

# ✨ Features

### ⚡ Smart Command System

* Structured command handling
* Fast asynchronous execution
* Modular architecture

### 📂 File Storage

* Store media in Telegram channels
* Instant retrieval system
* Organized indexing

### 🔎 Auto Filter

* Automatic search for stored files
* Pagination support

* Built with **aiohttp**

# ⚙️ Installation

## 1️⃣ Clone the Repository

```bash
git clone https://github.com/yourusername/gojo-assistant.git
cd gojo-assistant
```

---

## 2️⃣ Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 3️⃣ Configure Environment Variables

Set the required environment variables:

```
BOT_TOKEN=your_bot_token
API_ID=telegram_api_id
API_HASH=telegram_api_hash
DATABASE_URL=mongodb_connection_string
```

---

## 4️⃣ Run the Bot

```bash
python app.py
```
